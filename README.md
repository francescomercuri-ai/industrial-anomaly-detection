Unsupervised Surface Anomaly Detection for Industrial Castings

This repository implements an **unsupervised Deep Learning pipeline** for detecting surface anomalies in industrial manufacturing. 
Traditional geometric Computer Vision methods often fail in real-world scenarios due to lighting and texture variations. To solve this, this project leverages **Pre-trained CNN Feature Extraction (ResNet18)** coupled with a **K-Nearest Neighbors (KNN) Memory Bank**.
